先生の寄り道プロンプト帳

出張のごはん、職員室へのおみやげ、研究大会あとのすきま時間。仕事の「ついで」がちょっと楽しくなる、コピペで使えるGeminiプロンプト10本です。授業でも校務でもありません。それがいいんです。

対象：先生（人間のほう） 使うAI：Gemini（他のAIでも動きます） 所要：コピペ10秒
使い方は3ステップ
  1. 1.使いたいプロンプトの「コピー」ボタンを押す
  2. 2.[　] の中を自分の予定に書きかえて、Geminiに貼る
  3. 3.出てきた答えに、遠慮なく追い質問する（「雨なら？」「予算半分なら？」）
ひとつだけ、お約束 お店や施設の情報は変わります。営業時間・定休日・予約の要否は、出発前に公式サイトか電話で最終確認を。Geminiの答えは「候補出しの相棒」と割り切るのが、いちばん楽しい使い方です。
01SHUTCHO GOURMET

出張ごはん

出張の楽しみの8割はごはんです。異論は認めます。

№1昼休憩45分のランチ作戦

午後の部が始まる前に、サッと、でもせっかくなら地元のものを。

あなたは出張慣れした、地元通のグルメガイドです。

# 状況
出張で [市町村名・会場名] に来ています。
昼休憩は [45分] だけ。会場から徒歩 [5分] 圏内で探します。

# 条件
- 一人でサッと入れて、提供が早い店
- できればこの土地らしいもの（チェーン店は最終手段）
- 行列必至の有名店は外す
- 予算は [1,000円前後]

# 出力形式
候補を3つ、次の表で：
店名｜ジャンル｜会場からの徒歩時間｜おすすめ理由（1行）｜混雑リスク

最後に「時間が押したときの保険の1軒」も足してください。
※営業時間・定休日が不確かな場合は、その旨を正直に書くこと。

追い質問「会議が長引いて残り20分になった。プランBは？」

№2一人夜ごはん、地元の名物

泊まりの出張。ホテルのコンビニ飯で済ませたら、もったいない夜に。

あなたは一人めし・一人飲みの達人で、この土地の食に詳しい案内人です。

# 状況
出張で [市町村名] に泊まります。ホテルは [駅名・ホテル名] の周辺。
夜は自由時間。せっかくなので地元の名物が食べたい。

# 条件
- 一人でも入りやすい店（カウンター席がある、常連だけの空気じゃない）
- 予算は [3,000円] くらい
- お酒は [少し飲む／飲まない]
- 明日も仕事。21時台にはホテルに戻る

# 出力形式
1. 「この土地に来たらまずこれ」という名物料理を3つ、理由つきで
2. それが食べられる一人向きの店を、それぞれ1軒ずつ
3. 最後に「迷ったらここ」の第1推薦を1つ、断言してください

※営業時間・定休日は変わるので、公式情報での最終確認をすすめる一文も添えて。

追い質問「2軒目に軽く甘いものを食べるなら？」

№3帰りの駅弁、選んで

発車まで20分。駅弁売り場の前で固まる時間も含めて、出張です。

あなたは駅弁と車内グルメの評論家です。解説は軽妙に。

# 状況
[駅名] から [新幹線／特急] で帰ります。乗車時間は [約2時間]。
発車まで [20分]。駅の構内で調達します。

# 出力形式
- この駅で買うべき駅弁ベスト3（名前・値段の目安・推しポイント1行）
- 駅弁以外の「実はこっちが正解」枠を1つ
- 飲み物とお茶うけの組み合わせ提案も1つ

選んでいる時間まで楽しくなる解説をお願いします。
※商品の入れ替わりがありうることは正直に書いてください。

追い質問「車内で食べやすさ重視ならどれ？（においが強くないもの）」

02OMIYAGE

おみやげ

「個包装・日持ち・人数分」。職員室おみやげは、実は条件整理の問題です。

№4職員室おみやげ会議

30人分・個包装・切り分け不要。この三重条件を一発で解いてもらう。

あなたは「職場へのおみやげ選び」専門のコンシェルジュです。

# 状況
[都道府県・都市名] への出張帰り。職員室におみやげを買います。
買える場所は [駅／空港] の売店です。

# 絶対条件
- 職員室の人数：[30人]。全員に行き渡ること
- 個包装で、切り分け・お皿・冷蔵庫が不要
- 日持ちが1週間以上
- 予算：[2,000円以内]

# 出力形式
候補3つを表で：
商品名｜個数と価格の目安｜日持ち｜「これどこの？」と聞かれたときの返し（1行）

最後に、個数が足りないときの「2箱作戦」も添えてください。
※定番商品を優先し、実在があやしいものは出さないこと。

追い質問「甘いものが苦手な先生用に、しょっぱい枠を1つ足すなら？」

№5家族へのおみやげ＋30秒の小話

モノだけじゃなく「おみやげ話」も一緒に持って帰るためのプロンプト。

あなたは、渡す相手の顔を思い浮かべながらおみやげを選ぶプロです。

# 状況
[都道府県・都市名] への出張帰り。家族におみやげを買います。
渡す相手：[小3の娘と妻]（好み：[甘いもの好き・キャラクターもの好き] など）
買える場所は [駅／空港]、予算は合計 [2,000円]。

# 出力形式
1. 相手ごとにおすすめを2つずつ（品名と、その人に合う理由）
2. 渡すときの「30秒の小話」台本
   （「これはね、〇〇っていう土地でね…」から始まる、
     その土地の豆知識を1つ混ぜたひとこと話）

小話はウケ狙いすぎず、食卓でそのまま話せるトーンで。

追い質問「小話をもう少し面白く。オチを1つつけて」

03KENKYU-TAIKAI TRIP

研究大会のすきま時間

せっかく遠くまで来たんです。帰りの便までの2時間、無駄にしない。

№6帰りの便まで2時間、観光プラン

大会終了から出発まで。キャリーを引きずらない、乗り遅れない、でも満喫する。

あなたは「限られた時間で土地を味わう」プランニングの達人です。

# 状況
研究大会で [市町村名] に来ています。
[会場名・最寄り駅] を [15:30] に出て、[駅・空港名] を [18:00] の便で発ちます。
使えるのは正味 [2時間]。荷物は [キャリーあり（ロッカーに預けたい）]。

# 条件
- 移動は徒歩と公共交通のみ
- 「ここに来た」と思える場所を最低1つ入れる
- 乗り遅れリスクは絶対に取らない（駅・空港には出発30分前に着く）

# 出力形式
タイムライン形式で：
時刻｜場所・行動｜移動手段と所要時間｜ひとことメモ

プランは「王道コース」と「渋いコース」の2本立てでお願いします。

追い質問「雨が降ってきた。屋内版に組み替えて」

№7大会前の朝さんぽ

早起きは先生の職業病。どうせ目が覚めるなら、知らない街の朝を歩く。

あなたは早起きが得意な、まちあるきの案内人です。

# 状況
[市町村名] の [駅名・ホテル名] 周辺に泊まっています。
明日の朝、[6:30] から [1時間] だけ自由時間があります。

# 出力形式
- 朝の空気が気持ちいい散歩コースを1つ（往復の目安時間つき）
- 途中で寄れる朝ごはん・モーニングの店を1〜2つ
  （早朝から開いている店に限る。開店時間も書くこと）
- 「朝のうちに見ると得をする」この土地ならではの風景があれば1つ

戻る時刻から逆算して、無理のないコースにしてください。

追い質問「30分しか取れなくなった。短縮版は？」

№81時間しかない。1つに絞って

あれこれ調べる時間すらないとき。目利きに「これだけ」を断言してもらう。

あなたは「その土地の外せない1つ」を断言してくれる旅の目利きです。

# 状況
[市町村名] に来ましたが、自由時間は [1時間] だけ。
今いる場所は [駅名・会場名] のあたり。あれこれ回れません。

# 出力形式
1. 「迷わずこれ」を1つだけ断言（観光・食・体験、なんでもよい）
2. なぜそれか、3行で
3. 「次に来たときのお楽しみリスト」を3つ
   （今回は見送るが、覚えておく価値のあるもの）

複数の候補を並べて選ばせるのは禁止。あなたが決めてください。

追い質問「それ、行ってきた。次の30分でもう1つやるなら？」

04EVERYDAY

ふだん使い

出張がなくても使えます。金曜の夜と週末のために。

№9週末3時間のリフレッシュ処方箋

「休めるときに休む」も仕事のうち。プランニングごと外注しましょう。

あなたは、がんばりすぎる人に「休み方」を処方するプランナーです。

# 状況
今週末、[土曜の午後] に [3時間] だけ自由時間ができました。
自宅は [市町村名] の周辺。移動は [車／電車]。

# 希望
- 気分：[自然に癒されたい／おいしいもの／一人で静かに／体を動かしたい]
- 予算：[2,000円くらい]

# 出力形式
- 晴れプランと雨プランを1本ずつ、タイムラインで
- それぞれ「これをやったら帰る」という切り上げポイントを明示
  （ずるずる延ばして疲れて帰るのを防ぐため）

「何もしないで昼寝」が最適解だと思ったら、正直にそう書いてください。

追い質問「家族（[構成]）も一緒に行けるプランに変えて」

№10幹事になってしまった

歓送迎会、打ち上げ、忘年会。幹事の不安を段取りごと引き受けてもらう。

あなたは経験豊富な宴会幹事のアドバイザーです。

# 状況
[歓送迎会／打ち上げ／忘年会] の幹事になりました。
場所は [市町村名・駅名] の周辺、日程は [金曜19:00] を予定。

# 条件
- 人数：[20人]（出欠の変動あり。前日まで人数調整できる店が安心）
- 予算：一人 [4,000円]（飲み放題 [あり／なし]）
- そのほか：[個室希望／アレルギー対応が必要 など]

# 出力形式
1. この条件をグルメサイトで探すための「検索キーワード」1行
2. このエリア・この会の趣旨に合うおすすめジャンル3つと理由
3. 幹事の段取りチェックリスト
   （予約→事前アナウンス→当日→会計→お礼まで、時系列で10項目）

チェックリストには「幹事がいちばん忘れがちなこと」を1つ含めてください。

追い質問「乾杯挨拶を頼む人への依頼文も書いて（丁寧すぎない感じで）」

プロンプトは全部、書きかえて使ってかまいません。「45分」を「30分」に、「職員室」を「学年部」に。自分の学校仕様に育てていくのが、いちばんの正解です。よい寄り道を。
