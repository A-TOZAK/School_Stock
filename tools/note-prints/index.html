「先生、紙がありません」の、
その一枚に。

方眼・マス・作文。授業でよく使うノートを、そのまま印刷して配れる形にしました。ドリルの升目が合わない、書くスペースが足りない、そんなときの一枚に。全学年で使えて、ぜんぶ無料です。配布は note でしています。

●全学年対応 ●印刷するだけ ●登録・費用なし ●PDFで配布（note）
つかいかた
  1. 1使いたいノートのカードを開くと、note の記事に飛びます。
  2. 2記事の中の PDF をダウンロードして、そのまま印刷します。
  3. 3必要な枚数だけ刷って配るだけ。登録も費用もいりません。
