金属の温度と体積 ── 線路のつなぎ目のなぞ

単元「物の温度と体積（金属、水、空気と温度）」 ／ 本時：金属の温度と体積（全7時間中の第7時）

教科・学年理科・第4学年
本時のねらい金属も温度で体積が変わることを、実験からとらえる
時間45分（本時）

1　本時の目標

知識・技能：金属も、温めると体積が増え、冷やすと体積が減ることを、実験を通してとらえている。
思考・判断・表現：既習（空気・水）や生活経験をもとに、根拠のある予想をし、結果から考察して表現している。
主体的に学習に取り組む態度：身のまわりの金属（線路など）と関係づけて、進んで考えようとしている。

2　準備物

  • 球膨張実験器（金属球と輪）
  • アルコールランプ・マッチ・ぬれぞうきん
  • 湯（60℃程度）を入れたビーカー
  • 水を入れたビーカー（冷やす用）
  • 電車の走行音（つなぎ目の音）・つなぎ目の写真
  • ワークシート
  • （あれば）銅棒＋レンガの自作膨張器

3　本時の展開

温めると、金属の体積はどうなるのだろうか。
過程学習活動主な発問（T）と予想される反応（C）指導上の留意点・評価
つかむ10分 電車の音を聞き、つなぎ目のすき間に気づく。問題をつかむ。 T この音、何の乗り物かな。
C 電車！ ガタンゴトンって鳴ってる。
T この音、線路のつなぎ目のすき間で鳴っているんだよ。乗り心地が悪くなるのに、なぜわざわざすき間をあけているのかな。夏はどうなると思う？		 音→写真の順で提示し、身近な「なぜ？」を引き出す。
既習（空気・水は温めると体積が増える）を思い出させる。
予想7分 金属を温めると体積がどうなるか予想し、理由を書く。 C 固い金属は変わらないと思う。
C 空気や水と同じで、増えるんじゃない？
T どうしてそう思うの？ ノートに理由も書こう。		 予想が割れることを大切にする。
個人で書いた後、ペアで交流し、実験方法の見通しをもたせる。
【思・判・表】根拠のある予想を書いているか（記述）。
実験15分 球膨張実験器で確かめる。湯→炎→冷やす、の順で。 T まず湯で温めて、輪を通してみよう。
C …あれ、まだ通る。変わらないのかな。
T では、炎で60秒熱してみよう。
C 通らない！ 大きくなった！
T 水で冷やすと？
C また通った！		 湯では変化が小さく、炎で通らなくなる落差を体験させる。
やけど・アルコールランプの扱いに十分注意する。
【知・技】器具を正しく扱い、結果を記録している（行動観察・記述）。
使う8分 活用問題を考える。学んだ性質を使う。 T 熱していない今、通らない「B輪」がある。玉を通すにはどうすればいい？
C 玉を冷やす！ 玉が縮むから。
C 輪のほうを温める！ 穴が広がるから。		 「冷やす派」「温める派」に分けて予想→確かめる。
学んだ性質（温めると増える・冷やすと減る）を使えているかを見る。
まとめ5分 結論を書き、生活とつなぐ。 T 分かったことをまとめよう。線路のつなぎ目のすき間は、夏はどうなる？
C 金属が伸びてすき間が小さくなる。すき間がないと、線路が盛り上がってあぶない。		 結論を自分の言葉で書かせる。
電線・金属のふたなど、他の例にも広げる。
【態度】生活と関係づけて考えようとしている（発言・記述）。

4　板書計画

板書計画
問題温めると、金属の体積はどうなるのだろうか。　　【写真：線路のつなぎ目のすき間】
予想・変わらない（固いから）
・増える（空気・水と同じ）
実験の結果湯 → まだ通る／炎 → 通らない！／冷やす → また通る
考えたこと（考察）金属も、温めると体積が増える。冷やすと減る。
使ってみようB輪に玉を通すには？
→ 玉を冷やす／輪を温める
まとめ金属は、温めると体積が増え、冷やすと減る。だから夏は線路が伸びて、つなぎ目のすき間が小さくなる。
