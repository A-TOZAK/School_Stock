当事者に出会う ── 安心は、人によってちがう

単元「安心・安全なまちって、何だろう？（福祉×防災・全15時間）」 ／ 本時：第7時（当事者・専門家に出会う）

自作・無料
2026
教科・学年総合・第4学年
本時のねらい当事者の話から「安心は人によってちがう」に気づく
時間45分（本時・第7時／全15時間）

1　本時の目標

知識・技能：当事者の話を聞き、同じまちでも安心・安全は人によってちがうことに気づき、UD・バリアフリー・防災の考え方を理解し始めている。
思考・判断・表現：これまで調べて分からなかったことを軸に質問し、当事者の視点から自分たちの調査を捉え直している。
主体的に学習に取り組む態度：多様な立場の人の話を受け止め、まちをつくる一員として自分にできることを考えようとしている。

2　準備物

  • ゲスト（社会福祉協議会／車いす・視覚障害の当事者／市防災担当 等）※夏休み中に打診
  • これまでの白地図（安心マーク・心配マップ）
  • 国語でつくった質問カード
  • 記録用タブレット・ワークシート
  • お礼の準備（感謝は子ども・担任の自筆で）

3　本時の展開

当事者の人にとって、このまちは「安心・安全」だろうか。
過程学習活動主な発問（T）と予想される反応（C）指導上の留意点・評価
つかむ7分 これまでの調査（白地図）をふり返り、今日のめあてをつかむ。 T みんなが歩いて見つけた「安心」と「心配」。でも、それは"だれの"目で見た安心だったかな。
C ぼくたちの目…。
T 今日は、実際にまちで暮らす方に来ていただきました。		 子どもの調査を価値づけてから出会わせる。「調べて分からなかったこと」を思い出させる。
出会う18分 当事者・専門家の「一日」を聞く。質問する。 C（質問）駅の階段は、どうやって使っていますか。
C 大雨の日、逃げるとき困ることは？
C 避難所で、こまったことはありますか。		 当事者の一日を具体で語ってもらう。質問は国語でつくったカードを使う。「かわいそう」でなく「どうすれば」に向かう問い方を支える。
【思・判・表】分からなかったことを軸に質問しているか（発言）。
つなぐ12分 聞いたことを、自分たちの白地図・調査に重ねる。 T 今の話を聞いて、みんなの地図の「安心マーク」、そのままでいいかな。
C あそこ、ぼくには安全だけど、車いすの人には段差が…。
C 「安心」って、人によってちがうんだ。		 「安心は人によってちがう」を、教師が言うのでなく子どもの言葉から拾って板書する。ここが本時の山場。
【知・技】安心が人によって異なることに気づいているか（発言・記述）。
まとめ8分 ふり返りを書く。次時（できることを選ぶ）への見通し。 T 今日、いちばん心に残ったことと、次に調べたい・してみたいことを書こう。
C だれにとっても安心なまちって、まだできていないかも。
C わたしたちにできることを考えたい。		 「わかった」が「動きたい」に向くふり返りに。お礼は子ども・担任の自筆で（AIや定型文で代えない）。
【態度】一員として自分にできることを考えようとしているか（記述）。

4　板書計画

板書計画
めあて当事者の人にとって、このまちは「安心・安全」だろうか。　【白地図：安心マーク／心配マップ】
当事者の話（聞いたこと）・駅の階段／点字ブロックの上の自転車
・大雨の日の避難／避難所の段差・トイレ
気づいたこと（子どもの言葉）「同じ道でも、人によって安心がちがう」「だれにとっても安心、はまだ？」
地図を見直すぼくの「安心マーク」→ 車いすの人には「心配」かも
次にしたいことわたしたちにできることを考えたい
まとめ安心・安全は、人によってちがう。だれにとっても安心なまちを、みんなで考えていこう。

この指導案について。 この案は、School Stock（現役の小学校教員）が実際に組んでいる15時間の単元の第7時を、他の先生が使える形に一般化して自作したものです。探究の過程は文部科学省「総合的な学習の時間編」解説、当事者・防災の扱いは内閣府・文科省の防災／避難所バリアフリー資料に学びました。敬意を込めて。先行事例と出典は 教材研究ノートの記事 にまとめています。
